Condition

Condition can be used to make sure that something is true or false before proceeding to the next part. In order to declare a Condition in the PL/0 language, there should either be an Expression or an odd first declared.

If an Expression is used first, then it needs to be followed by a relational operation and another Expression. A relational operation can either be one of these symbols: "="|“<>"|"<"|"<="|">"|">=“. These symbols stands for equal, not equal, less than, less than or equal, greater than, or greater than or equal to, respectively. By combing these components, then one can write their own Condition that compares a combination two variables/ constants.

For instances:

  • X > 2
  • Y <= 4
  • Z <> 8
